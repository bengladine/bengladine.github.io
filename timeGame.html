About me

Time Game

The goal

Creating a game where Time Manipulation stands central. Specifically manipulating time on specific objects. We'd like to find a way to use this mechanic in a Player vs Player way.

The process

I first got the idea after watching the movie Tenet for the second time. Wouldn't it be cool to manipulate objects in time and actually use that in a combat style way? So I started with the initial concept. Reverting an object in motion. Time Data had to be gathered, which is the position, the rotation and the velocity of an object. This data was then recorded on every object that was considered a time object. After that, you as the user would look at the object, press a button and it would revert! See the gifs below for some examples of that.

My job and take aways

I'm working on the main mechanic, namely the manipulation of time on objects. Then pouring this mechanic into a gun that controls this. Next to that I'm working on making guns that can also be fired and its projectiles be time manipulated as well.

Aside from that, taking part of the game design through brainstorms, dicussions, testing sessions, etc, ...
